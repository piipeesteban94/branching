Pasos para crear un repositorio

  • Paso 1: usar git init para iniciar repositorio local
  • Paso 2: agregar los arcivos que se utilizaran en el repositorio con el comando git add
  • Paso 3: generar el primer commit con el comando git commit -m "(inserte aqui el texto)"
  • Paso 4: verificar con el comando git status si hay que guardar o no una nueva version
